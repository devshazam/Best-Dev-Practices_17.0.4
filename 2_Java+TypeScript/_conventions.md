# 1 Naming Convention:
    - Variables
        - Числовая или строковая перемнная
            - camelCase: dogName
            + noun: dogName, dogAge 
        - Boolean
            - camelCase: isTrained
            + вопрос: dogName, dogAge 
        - Константа
            - UPPER_SNAKE_CASE: DAYS_IN_WEEK
        - Функция
            - camelCase: dogName
            + verb: getDogName, setName
